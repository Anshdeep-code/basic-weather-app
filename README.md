# 🌤️ Weather App

A simple and responsive Weather App built using **HTML**, **CSS**, and **Vanilla JavaScript**. It fetches real-time weather information using the OpenWeather API and displays the current weather conditions for any city entered by the user.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature
- ☁️ Weather condition and icon
- 💧 Humidity
- 🌬️ Wind Speed
- 🌡️ Atmospheric Pressure
- 👁️ Visibility
- ❌ Error message for invalid city names
- 🎨 Modern and responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Anshdeep-code/basic-weather-app.git
```

2. Navigate to the project

```bash
cd weather-app
```

3. Open `index.html` using Live Server or any local development server.

---

## 🔑 API Setup

This project uses the **OpenWeather API**.

1. Create a free account at:
   https://openweathermap.org/api

2. Generate an API key.

3. Open `script.js` and replace:

```javascript
const API_KEY = "OPENWEATHER_API_KEY";
```

with your own API key.
